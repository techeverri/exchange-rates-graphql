#### Query

```graphql
query CountryQuery($name: String!, $base: String!) {
  country(name: $name) {
    name
    population
    currencies {
      name
      code
      symbol
      rate(base: $base)
    }
    flag
  }
}
```

#### Query Variables

```json
{
  "name": "Colombia",
  "base": "SEK"
}
```
